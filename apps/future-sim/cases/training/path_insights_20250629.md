# 路径洞察 — 2025-06-29

## 已识别路径类型分布

| 路径类型 | 样本数 | 代表案例 | 关键特征 |
|---------|--------|---------|---------|
| **breakout** | 3 | Lovable, Chatbase, Flodesk | 强痛点 + 趋势对齐 + 多渠道分发 |
| **niche_compound** | 3 | Postiz, Papermark, Tower | 差异化 + 留存强 + 社区驱动 |
| **strong_pain_tool** | 3 | Post Bridge, AudioPen, MonAi | 痛点强 + 传播不一定强 |
| **long_compound** | 2 | Photopea, Stackby | 前期慢 + 留存强 + 口碑积累 |
| **short_burst_decay** | 1 | Kleo | 短期爆发后衰退 |
| **monetization_failed** | 1 | marblism | 有用户但商业化失败 |

## 关键发现

### 成功路径的共性
1. **痛点强度是最强预测变量** — 所有成功案例都有明确的用户痛点
2. **差异化是第二强预测变量** — 开源(Postiz)、非技术用户(Lovable)、免费(Photopea)
3. **趋势对齐加速增长** — Lovable/Chatbase 都踩中 AI 工具爆发期
4. **迭代速度决定存活** — 成功案例都在快速迭代

### 失败路径的共性
1. **分发弱是最常见死因** — 产品好但没人知道
2. **留存低导致快速衰退** — 高曝光低留存 = 死亡螺旋
3. **商业化不匹配** — marblism 有用户但无法有效变现
4. **单一渠道依赖** — 被一个渠道绑架 = 高风险

### 变量与路径类型的相关性

| 变量 | 最强相关路径 | 方向 |
|------|------------|------|
| pain_intensity | breakout | 正相关 |
| clarity | breakout | 正相关 |
| differentiation | niche_compound | 正相关 |
| distribution_power | breakout | 正相关 |
| retention_design | long_compound | 正相关 |
| monetization_fit | moderate_success | 正相关 |
| competition_pressure | dead | 正相关 |
| technical_debt | dead | 正相关 |

### 样本不足的路径类型
- **low_exposure_death** — 需要更多失败案例
- **competition_crushed** — 需要被竞品压制的案例
- **technical_debt_collapse** — 需要技术债导致死亡的案例
- **abandoned_by_creator** — 需要创始人放弃的案例

### 最缺失的阶段数据
- **pre_launch → day_1** — 几乎没有首日数据
- **day1_retention / day7_retention** — 公开数据极少
- **user_growth_curve** — 需要月度增长数据
- **churn_rate** — 几乎无公开数据
- **acquisition_cost** — 无公开数据

### 评分规则调整建议
1. **distribution_power 权重可能被高估** — 很多成功案例初期分发弱
2. **retention_design 权重可能被低估** — 长期复利案例留存设计是关键
3. **pain_intensity 需要更细粒度** — 强痛点但弱传播 = 不同路径
4. **competition_pressure 需要区分** — 竞争强但有差异化 ≠ 竞争强无差异化
