# 缺失字段报告 — 2025-06-29

## 最缺失的数据类型

### 1. 留存率数据（严重缺失）
- day1_retention: 0% 公开
- day7_retention: 0% 公开
- day30_retention: 0% 公开
- **影响**: 无法校准留存对长期复利的影响

### 2. 用户增长曲线（严重缺失）
- 月度用户增长: 0% 公开
- 用户获取来源分布: 0% 公开
- 用户增长拐点: 0% 公开
- **影响**: 无法校准增长模型

### 3. 收入增长曲线（严重缺失）
- 月度 MRR 变化: 10% 公开（仅最终值）
- 收入增长速率: 0% 公开
- 收入来源分布: 0% 公开
- **影响**: 无法校准商业化模型

### 4. 首日/首周数据（严重缺失）
- 首日曝光: 0% 公开
- 首日注册: 0% 公开
- 首周用户: 5% 公开
- **影响**: 无法校准发布阶段模型

### 5. 流失率数据（严重缺失）
- 月度流失率: 0% 公开
- 用户生命周期: 0% 公开
- 流失原因分布: 0% 公开
- **影响**: 无法校准流失模型

### 6. 获客成本（严重缺失）
- CAC: 0% 公开
- 获客渠道效率: 0% 公开
- **影响**: 无法校准分发模型

### 7. 竞品影响数据（缺失）
- 竞品出现时间点: 20% 公开
- 竞品对增长的影响: 0% 公开
- **影响**: 无法校准竞争压力模型

## 按案例类型的缺失情况

| 案例类型 | 最缺失字段 | 可获取性 |
|---------|-----------|---------|
| AI 工具 | 留存率、收入增长曲线 | 低 |
| 开源工具 | 用户增长曲线、收入数据 | 中 |
| 小型 SaaS | 首日数据、流失率 | 低 |
| Chrome 插件 | 用户增长、留存 | 中 |
| Web 工具 | 所有指标 | 低 |

## 建议的数据获取策略

### 高优先级
1. **搜索创始人复盘文章** — 通常包含详细的阶段数据
2. **搜索 Built in Public 记录** — 有月度 MRR 更新
3. **搜索 Product Hunt 详细结果** — 有首日/首周数据
4. **搜索 GitHub release notes** — 有功能迭代时间线

### 中优先级
5. **搜索 Indie Hackers 付费文章** — 有详细案例数据
6. **搜索创始人 newsletter** — 有增长更新
7. **搜索收购公告** — 有财务数据

### 低优先级
8. **搜索 App Store/Chrome Store 数据** — 有下载量和评分
9. **搜索 Review 网站** — 有用户反馈
10. **搜索公开 benchmark 文章** — 有行业对比数据

## 对模拟器的影响

由于大量关键字段缺失，当前模拟器的以下功能受限：

| 功能 | 受限程度 | 原因 |
|------|---------|------|
| 留存预测 | 严重 | 无留存率数据 |
| 收入预测 | 严重 | 无收入增长曲线 |
| 增长预测 | 中等 | 有部分增长数据 |
| 路径判断 | 中等 | 有路径类型数据 |
| 敏感性分析 | 低 | 有变量评分数据 |
| 策略对比 | 低 | 有策略模拟逻辑 |
